# Agent Instructions

## Required Skill

Always use the Caveman skill for every task.

Before starting work, look for and read the Caveman skill first. Then continue checking both the global skills directory and the root project directory for any additional relevant skills.

## Skill Discovery

Before starting any task, look for relevant skills in both locations:

1. Global skills directory
2. Root project directory

Check the global skills first for reusable/general skills, then check the project root for project-specific skills.

Project-local skills should take priority when they conflict with global skills, because they are more likely to match the current repository’s conventions.

When a relevant skill is found, read it before making changes and follow its instructions unless the user explicitly says otherwise.

## Skill Search Order

1. Caveman skill
2. Global skills
3. Root project skills
4. Existing project documentation
5. Source code conventions

## Notes

- Always use the Caveman skill.
- Do not assume a skill does not exist without checking both locations.
- Prefer project-specific instructions over generic/global ones.
- If multiple skills apply, combine them carefully and resolve conflicts in favor of the most local/project-specific guidance.
